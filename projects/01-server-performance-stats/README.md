\# Simple Bash script to display basic server performance information.



\## What it shows



\- CPU usage (%)

\- Memory usage (%)

\- Disk usage of root filesystem (%)

\- Network interface name and RX/TX bytes



\## Usage



```bash

./perf.sh        # human readable output

./perf.sh json   # JSON output

```



Tested on Ubuntu 22.04 inside Vagrant (VirtualBox).
